Select one of the selection in each section's buttons below to choose an alterntive link as a default.

Input a number that is listed for the link you want. Any invalid link will not work.


The selected link will save for the next visit. You can also input a custom link which is an advanced option.

This is to bypass blocks from the official links with official and unoffical alternative links.

It is important to have a backup proxy around so you can get to the main page as if the main site is blocked, this won't work anymore.

Technically, you could use the proxy to get the alternative links but it's up to you.



If you want your link added here or to the custom use button, clone the Assets/Villade/Revite repositories and host them with Github Pages, Vercel, Netlify, etc.

Then just get the url of the main page of either Assets, Villade, or Revite and input it into the custom link field. The links will automatically load from your mirror.
